## Trade History
###  1. Spot Trade History
#### Response Parameters
| Parameter  | Type | Comments|
|:----- |:-------|:-----|
OrderID| string| Order ID of the closing order|
TradeID| string| Trade ID of the closing order|
Symbol| string| Name of the trading pair in the form of|
Side| string| Side. Either Buy or Sell direction|
QuoteCoin| string| Symbol of the Quote Coin|
BaseCoin| string| Symbol of the Executed Coin|
ExecPrice| string| Execution Price|
ExecValue| string| Transacted volume|
TradingFee| string| Trading fee for a single fill|
FeeToken| string| Token of the Trading Fee|
TradeTime| string| UNIX Time. This is in milliseconds. Time when the trade was executed|

## Bonus History

:::tip
<ul><li><b>Bonus</b> - The experience bonus can be used to open a position and deduct the handling fee and the funding rate fee because it is paid directly to the user's asset account. All bonuses shown are already applied by the users.</li></ul>
<ul><li><b>Coupon</b> - Money from coupons are deductible will only be used to offset the handling fee. It will not be added to the user's asset account. It will be refunded to the user's asset account following the deduction. All coupons shown are already applied by the users.</li></ul>
:::
